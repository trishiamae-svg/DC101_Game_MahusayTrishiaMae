TITLE:
Typing Speed Test

DESCRIPTION:
The Typing Speed Test is a simple web-based game that measures how fast and accurate a user can type. The player is given a word and must type it correctly within a given time. This game helps improve typing skills and is useful for students, beginners, and anyone who wants to practice typing.

TECHNOLOGIES USED:
HTML – for the structure of the game interface
CSS – for styling and layout design
JavaScript – for game logic, timer, accuracy checking, and WPM calculation

INSTRUCTION TO RUN THE GAME:
1. Create a folder for the project.
2. Inside the folder, add the following files:
- index.html
- style.css
- script.js
3. Open index.html using any web browser (Google Chrome, Edge, or Firefox).
4. Click Start to begin the typing test.
5. Type the given text as accurately and quickly as possible.
6. When the timer ends, view your score.
